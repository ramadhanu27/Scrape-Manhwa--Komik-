║         SHINIGAMI MANHWA SCRAPER - QUICK START GUIDE          ║
╚═══════════════════════════════════════════════════════════════╝

📁 STRUKTUR FOLDER:
├── scrapers/          → Script untuk scraping
├── website/           → Website files (index.html, dll)
├── data/              → Data hasil scraping
├── chapters/          → Metadata chapter
├── images/            → Gambar yang di-download
└── docs/              → Dokumentasi lengkap

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 LANGKAH-LANGKAH:

1️⃣  SCRAPE LIST MANHWA
   cd scrapers
   node scrape-real-images.js 5
   
   → Scrape 5 halaman (120 manhwa)
   → Output: data/manhwa-all.json

2️⃣  DOWNLOAD CHAPTER IMAGES
   cd scrapers
   node scrape-chapters.js 1 2
   
   → Download 1 manhwa, 2 chapter
   → Output: chapters/ dan images/chapters/

3️⃣  BUILD WEBSITE
   node build-all.js
   
   → Build index.html
   → Embed data ke manhwa-detail.html
   → Siap dibuka langsung!

4️⃣  BUKA WEBSITE
   Buka: index.html (di root folder) di browser
   
   ⚠️  PENTING: Jangan buka dari folder website/!
   
   → Klik "Read Chapters"
   → Pilih chapter
   → Baca manhwa!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 PARAMETER SCRAPER:

scrape-real-images.js [pages]
   - 5  = 120 manhwa
   - 10 = 240 manhwa (semua)

scrape-chapters.js [manhwa] [chapters]
   - 1 2      = 1 manhwa, 2 chapter
   - 5 5      = 5 manhwa, 5 chapter each
   - 10 10    = 10 manhwa, 10 chapter each
   - 1 all    = 1 manhwa, SEMUA chapter
   - 10 all   = 10 manhwa, SEMUA chapter each
   - 120 all  = SEMUA manhwa, SEMUA chapter

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  TROUBLESHOOTING:

❌ "Chapters Not Found"
   → Belum download chapter
   → Run: cd scrapers && node scrape-chapters.js 1 2

❌ "Failed to fetch"
   → Path salah atau file tidak ada
   → Pastikan buka dari folder website/

❌ Gambar tidak muncul
   → Cek folder images/chapters/
   → Pastikan download selesai

✓ Monitor disk space (1 chapter ≈ 5-15 MB)
✓ Download bertahap untuk koneksi lambat
✓ Backup folder chapters/ dan images/
